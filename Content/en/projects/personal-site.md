# This site

**Stack:** Swift · Publish · Plot
**Link:** [github.com/JohnSundell/Publish](https://github.com/JohnSundell/Publish)

The bilingual card site with a blog and portfolio you're reading right now.
Built with [Publish](https://github.com/JohnSundell/Publish) — a static site
generator in Swift — with a custom theme on [Plot](https://github.com/JohnSundell/Plot).

Content is written in Markdown, the markup is described type-safely in Swift,
and the output is static HTML without a single line of JavaScript.
